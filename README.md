# ⚙️ Mendata - Soluções Inteligentes

A **Mendata** é uma consultoria especializada em transformar dados e processos em resultados reais. Este site institucional foi desenvolvido por **Camila Mendes** para apresentar soluções personalizadas de automação, logística e análise estratégica, unindo conhecimento técnico à vivência operacional.

🔗 **Site ao vivo:** [https://www.mendata.com.br/](https://www.mendata.com.br/)

---

## 🚀 Sobre o Projeto
O objetivo da Mendata é converter tarefas manuais em soluções digitais inteligentes. O site reflete essa proposta através de uma interface moderna, focada em clareza visual para a tomada de decisão e otimização de rotinas.

## 🛠️ Tecnologias e Ferramentas
* **HTML5**: Estruturação semântica e SEO.
* **CSS3 Avançado**: 
  - Uso de **Flexbox** e **Grid** para layout responsivo.
  - Animações interativas e efeitos de transição suave.
  - Variáveis CSS para consistência visual.
* **GitHub Pages**: Hospedagem e deploy contínuo.

## 📸 Funcionalidades e Diferenciais
* **Interface Slim & Moderna**: Design otimizado para uma navegação fluida.
* **Responsividade Total**: Experiência perfeita em desktops, tablets e smartphones.
* **Seção de Portfólio**: Exposição estratégica de serviços e cases.
* **Conversão Direta**: Botão flutuante de WhatsApp e CTAs (Call to Action) estratégicos.
* **Identidade Visual**: Logo personalizada e paleta de cores focada em tecnologia.
